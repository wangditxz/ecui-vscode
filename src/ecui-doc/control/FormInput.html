FormInput 控件

表单项控件。

继承自 Control

文件路径：.../Users/yiche/yiche/ECUI/src/input/form-input.js

详情

用于将表单项控件化以重新定义相关的 getFormName/getFormValue/isFormChecked 等方法，是 InputControl 的简化版。

option支持的属性

name 输入框的名称

属性

_sName

方法

$disable 控件失效，阻止输入框提交。
$enable 控件解除失效，需要将输入框设置为可提交。
getFormName 获取控件进行提交的名称。
getFormValue 获取控件进行提交的值。
saveToDefault 保存控件的值为默认值，直接忽略。
