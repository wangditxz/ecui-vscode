Label 控件

标签控件。

继承自 Control

文件路径：.../Users/yiche/yiche/ECUI/src/input/label.js

使用方法

<div ui="type:label;for:checkbox"></div>
或
<div ui="type:label">
    <input ui="type:text" />
</div>

详情

将事件转发到指定的控件上，通常与 Radio、Checkbox 等控件联合使用，扩大点击响应区域。

option支持的属性

for 被转发的控件 id

属性

_cFor

方法

$click 鼠标单击控件事件的默认处理。
setFor 设置控件的事件转发接收控件。
