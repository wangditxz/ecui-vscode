TreeView 控件

树控件。

继承自 Control

文件路径：.../Users/yiche/yiche/ECUI/src/base/tree-view.js

使用方法

<ul ui="type:tree-view;collapsed:false">
    <li>根节点</li>

    <li>子节点一</li>
    <li>子节点二</li>
    <ul class="ui-hide">
        <li>子节点三</li>

        <li>孙节点一</li>
        <li>孙节点二</li>
        <li>孙节点三</li>
    </ul>

详情

包含普通子控件或者子树视图控件，普通子控件显示在它的文本区域，如果是子树视图控件，将在专门的子树视图控件区域显示。子树视图控件区域可以被收缩隐藏或是展开显示，默认情况下点击树视图控件就改变子树视图控件区域的状态。

option支持的属性

collapsed 子树区域是否收缩，默认为展开

属性

_bCollapsed
_eContainer
_aChildren

方法

$click 控件点击时改变子树视图控件的显示/隐藏状态。
$collapse 收缩当前树视图控件的子树区域的默认处理。
$dispose
$expand 展开当前树视图控件的子树区域的默认处理。
$hide 隐藏树视图控件的同时需要将子树区域也隐藏。
$mouseout
$mouseover
$nodeclick 节点点击事件。
$nodeout 节点移出事件。
$nodeover 节点移入事件。
$setParent 树视图控件改变位置时，需要将自己的子树区域显示在主元素之后。
$show 显示树视图控件的同时需要将子树视图区域也显示。
add 添加子树视图控件。
collapse 收缩当前树视图控件的子树区域。
expand 展开当前树视图控件的子树区域。
forEach 遍历所有的树控件节点。
getChild 获取当前树视图控件的指定子树视图控件。
getChildren 获取当前树视图控件的所有子树视图控件。
getRoot 获取当前树视图控件的根控件。
isCollapsed 当前子树区域是否收缩。
remove 获取当前树视图控件的所有子树视图控件。
