CheckboxTree 控件

复选框树控件。

继承自 TreeView

文件路径：.../Users/yiche/yiche/ECUI/src/input/checkbox-tree.js

构造方法中没有call()

使用方法

<ul ui="type:checkbox-tree;name:name">
    <li>根节点</li>

    <li>子节点一</li>
    <li>子节点二</li>
    <ul class="ui-hide" ui="subject:true">
        <li>子节点三</li>

        <li>孙节点一</li>
        <li>孙节点二</li>
        <li>孙节点三</li>
    </ul>
</ul>

详情

树的每一个节点都包含一个复选框标识选中状态，除非特别的使用 subject 参数指定，否则子节点与父节点的复选框缺省设置联动。

option支持的属性

name 复选框对应表单项的名称
value 复选框对应表单项的值
subject 如果为 true 表示与所有子节点的复选框进行联动，其它等价 false 的值表示不联动

属性

_uCheckbox

方法

getChecked 获取包括当前树控件在内的全部选中的子树控件。
getValue 获取当前树控件复选框的表单项的值。
isChecked 判断树控件是否选中。
setChecked 设置当前树控件复选框选中状态。
